## 安装部署方式

### 1. 一键安装脚本
```bash
wget https://raw.githubusercontent.com/your-repo/lnmp-panel/main/deploy/full_install.sh
chmod +x full_install.sh
sudo ./full_install.sh
```

### 2. 手动安装
```bash
# 下载二进制文件
wget https://github.com/your-repo/lnmp-panel/releases/latest/lnmp-panel-armv7l
sudo mv lnmp-panel-armv7l /usr/local/bin/lnmp-panel
sudo chmod +x /usr/local/bin/lnmp-panel

# 创建系统服务
sudo systemctl enable lnmp-panel
sudo systemctl start lnmp-panel
```

### 3. Docker部署
```bash
docker run -d \
  --name lnmp-panel \
  -p 8080:8080 \
  -v /etc/nginx:/etc/nginx \
  -v /etc/mysql:/etc/mysql \
  -v /etc/php:/etc/php \
  -v /var/log:/var/log \
  your-repo/lnmp-panel:armv7l
```

## API接口文档

### 核心API端点
- `GET /api/services` - 获取服务状态
- `POST /api/services/{name}/start` - 启动服务
- `POST /api/services/{name}/stop` - 停止服务
- `POST /api/services/{name}/restart` - 重启服务
- `GET /api/config/{name}` - 获取配置
- `POST /api/config/{name}` - 更新配置
- `GET /api/logs/{name}` - 获取日志
- `GET /api/system/info` - 系统信息
